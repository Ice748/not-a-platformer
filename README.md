# Controls 🕹️
| Key | Action |
| :--- | :--- |
| <kbd>W</kbd> <kbd>A</kbd> <kbd>S</kbd> <kbd>D</kbd> | Movement |
| <kbd>Space</kbd> | Jump |
| <kbd>Mouse</kbd> | Camera Rotate |
| <kbd>R</kbd> | Restart |
